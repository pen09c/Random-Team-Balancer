# ⚖️ Minecraft Random-Team-Balancer (Skript)

마인크래프트 서버에서 플레이어를 두 팀으로 **수학적으로 완벽하게 배분**하는 최적화된 스크립트입니다.

## ✨ 주요 특징
* **인원 균형 최적화:** `ceil()` 함수를 사용하여 전체 인원이 홀수일 때도 팀 간 인원 차이를 최대 1명으로 제한합니다.
* **지역 변수 사용:** `{_count}` 등 로컬 변수를 적극 활용하여 서버 메모리 누수를 방지하고 데이터를 깔끔하게 관리합니다.
* **랜덤 로직:** 중복 없는 무작위 추출 방식을 통해 매번 공정한 팀 배정을 보장합니다.

## 🚀 사용법
1. 스크립트 파일을 `plugins/Skript/scripts` 폴더에 넣습니다.
2. 서버에서 `/sk reload [파일명]`을 입력합니다.
3. `//team` 명령어를 입력하여 팀 배정을 시작합니다.

## 🛠️ 코드 로직 설명
1. **인원 파악:** 현재 접속 중인 모든 플레이어를 리스트에 담고 총 인원수를 계산합니다.
2. **팀 정원 설정:** 총 인원의 절반(올림 처리)을 Team 1의 정원으로 설정합니다.
3. **무작위 배분:** 루프를 돌며 랜덤 정수를 생성, 남은 정원에 따라 Team 1 또는 Team 2로 플레이어를 할당합니다.
4. **확장성:** 코드 하단의 `loop all players` 구문에 팀별 텔레포트나 메시지 기능을 자유롭게 추가할 수 있습니다.

## ⚠️ 요구 사항
* **Minecraft Version:** 1.12.2 ~ 최신 버전 호환
* **Skript:** 2.6 이상 권장
